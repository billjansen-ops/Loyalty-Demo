---
name: Stop spiraling on failed fixes
description: When a fix fails, stop and diagnose the root cause instead of trying more fixes
type: feedback
---

When a fix doesn't work, STOP. Do not try another fix immediately. Diagnose the root cause first.

**Why:** Session 98 — made 10+ failed attempts to fix a broken dashboard, each one making things worse. Bill had to tell me to stop multiple times. The root cause was simple (wrong codebase) but I never paused to figure that out.

**How to apply:** If a fix fails, read the error carefully, understand WHY it failed, then fix the actual cause. One attempt. If that fails too, stop and tell Bill what's happening honestly instead of trying again blindly.
